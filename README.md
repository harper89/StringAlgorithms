# String Algorithm Challenges

All of your work should be done in a single file named **`StringAlgorithms.java`**, which must also include a **`main`** method that tests each algorithm at least once.

## ‚úîÔ∏è Challenge 2 ‚Äî 'reverseWord'
Write a method **'reverseWord'** that takes a 'String word' as a parameter.  
The method should **return a new String with the characters reversed**.

**Example:**  
'reverseWord("hello")' ‚Üí '"olleh"'

---

## ‚úîÔ∏è Challenge 3 ‚Äî 'capitalizeString'
Write a method **'capitalizeString'** that takes a 'String sentence' as a parameter.  
The method should **return a new String in which the first letter of every word is capitalized**, leaving the rest of the letters unchanged.

**Example:**  
'capitalizeString("the quick brown fox")' ‚Üí '"The Quick Brown Fox"'

---

## ‚úîÔ∏è Challenge 4 ‚Äî 'detectPalindrome'
Write a method **'detectPalindrome'** that takes a 'String word' as a parameter.  
The method should **return 'true' if the word reads the same forward and backward** (ignoring case), and **'false' otherwise**.

**Example:**  
- 'detectPalindrome("Racecar")' ‚Üí 'true'  
- 'detectPalindrome("Java")' ‚Üí 'false'

---

## ‚úîÔ∏è Challenge 5 ‚Äî 'firstUniqueChar'
Write a method **'firstUniqueChar'** that takes a 'String word' and returns the **first character that appears exactly once** in that String.  
If no such character exists, return a space '' '' (or another sentinel value of your choice).

**Examples:**  
- 'firstUniqueChar("swiss")' ‚Üí ''w''  
- 'firstUniqueChar("aabbcc")' ‚Üí '' ''
